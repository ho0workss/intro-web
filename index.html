❓ 자주 묻는 질문

Q. 연차는 어떻게 신청하나요?

홈 → 휴가 관리 → "+ 휴가 신청" 버튼을 통해 신청할 수 있습니다. 첨부서류(PDF/DOC) 업로드가 필요합니다.

Q. 메신저에서 파일을 어떻게 보내나요?

메신저 입력창에 파일을 드래그&드롭하거나, 클립보드에서 붙여넣기(Ctrl+V)할 수 있습니다.

Q. 회의록에 표를 추가하려면?

회의 상세 페이지에서 /를 입력 후 "스프레드시트"를 선택하세요. 구글시트 수준의 함수와 병합 기능이 지원됩니다.

Q. AI 모델을 추가하려면?

관리자 계정에서 설정 → AI API 설정 메뉴에서 새 모델과 API 키를 추가할 수 있습니다.

Q. 비밀번호를 변경하려면?

설정 → 프로필 메뉴에서 변경 가능합니다.

🔌 API 연동

📘 가이드
· 네이버 스마트스토어: 셀러센터 → API 신청 → Client ID/Secret 발급
· 쿠팡: Wing → 판매자 마이페이지 → OpenAPI 사용자 등록 → Access/Secret Key
· 카페24: 운영자 모드 → 앱 설정 → OpenAPI 키 발급
· 네이버 검색광고: 광고 시스템 → 도구 → API 사용 설정
· Google Ads / Meta Ads: OAuth 인증 필요 (별도 안내 예정)
· 등록한 키는 브라우저 localStorage에 저장됩니다 (실배포 시 서버 환경변수로 이전 권장).
